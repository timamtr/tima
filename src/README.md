# Public Transport Management System (Assignment 2)

Данный проект представляет собой систему управления общественным транспортом, разработанную в рамках учебного задания. Программа позволяет управлять парком транспортных средств, выполнять поиск, сортировку и фильтрацию данных.

## Выполненные требования
В проекте реализованы следующие принципы ООП:
* **Абстракция**: Использование абстрактного класса `Transport` как основы для всех видов транспорта.
* **Инкапсуляция**: Все поля классов (`id`, `capacity`, `name` и др.) скрыты (private) и доступны через геттеры и сеттеры.
* **Наследование**: Классы `Bus` и `ExpressBus` наследуются от базового класса `Transport`.
* **Полиморфизм**: Переопределение метода `getTransportType()` в дочерних классах.
* **Переопределение методов**: В классах `Transport` и `Commuter` переопределены стандартные методы `toString()`, `equals()` и `hashCode()`.

## Функционал
Программа поддерживает следующие операции (реализованы в классе `TransportService`):
1. **Организация пула данных**: Хранение списка транспортных средств в `ArrayList`.
2. **Сортировка**: Сортировка транспорта по его вместимости (`capacity`).
3. **Поиск**: Поиск транспортного средства по уникальному идентификатору (`id`).
4. **Фильтрация**:
    - Выборка только экспресс-автобусов (`ExpressBus`) с использованием Java Streams и `instanceof`.
    - Выборка транспорта с вместимостью выше заданного порога.

## Структура проекта
* `Transport.java` — абстрактный родительский класс.
* `Bus.java` — класс обычного автобуса.
* `ExpressBus.java` — класс экспресс-автобуса.
* `Commuter.java` — класс, описывающий пассажира.
* `TransportService.java` — сервис для управления автопарком (бизнес-логика).
* `Main.java` — точка входа в программу, демонстрирующая работу всех функций.

## Как запустить
1. Скомпилируйте файлы: `javac *.java`
2. Запустите программу: `java Main`

# Public Transport Management System (Assignments 2 & 3)

Система управления общественным транспортом, позволяющая управлять автопарком, пассажирами и сохранять данные в реляционной БД.

## Технологии
* **Java 17+**
* **PostgreSQL** (DBMS)

## Выполненные требования (Assignment 2)
В коде реализованы ключевые принципы объектно-ориентированного программирования:
* **Абстракция**: Использован абстрактный класс `Transport` как фундамент для всех видов транспорта.
* **Инкапсуляция**: Все поля (`id`, `capacity`, `name` и др.) скрыты (private) и доступны через геттеры и сеттеры.
* **Наследование**: Классы `Bus` и `ExpressBus` расширяют функционал `Transport`.
* **Полиморфизм**: Метод `getTransportType()` возвращает уникальную строку для каждого подкласса.
* **Методы Object**: Переопределены `toString()`, `equals()` и `hashCode()` для корректного сравнения объектов.
* **Data Processing**: Использование Java Streams для фильтрации только экспресс-маршрутов.

## Интеграция с базой данных (Assignment 3)
Реализовано хранение данных в PostgreSQL:
* **Схема данных**: Созданы таблицы `transports` и `commuters`, повторяющие структуру Java-классов.
* **CRUD операции**:
   * **Create**: Метод `addTransport` добавляет данные через `PreparedStatement`.
   * **Read**: Метод `getAllTransports` выводит весь список транспорта из БД.
   * **Поиск**: Реализован поиск по `id` в `TransportService`.

## Как запустить
1. Создайте базу данных в Postgres и выполните команды из `Query.sql`.
2. Настройте подключение (URL, USER, PASSWORD) в классе `DatabaseManager.java`.
3. Запустите `Main.java` для проверки работы поиска и фильтрации.
